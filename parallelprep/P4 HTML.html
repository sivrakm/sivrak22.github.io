üìò What is HTML?

HTML stands for HyperText Markup Language.
HTML tells the browser what content to show and how it is organized.

üß† Think of HTML like the blueprint or skeleton of a webpage — it provides structure and meaning.

Example:

<h1>This is a heading</h1>
<p>This is a paragraph of text.

üîß HTML Page Structure

Here’s the basic structure of an HTML document:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is a simple webpage.</p>
  </body>
</html>

üß† What is <head>?

  • The <head> tag contains information about the page (not visible to the user).
  • It's where you include:
    • The page <title>
    • <meta> tags
    • Links to CSS files
    • Fonts and scripts

Example:

    <title>My Awesome Site</title>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />

    <meta name="description" content="Learn HTML the easy way!" />
    <meta name="keywords" content="HTML, Web development, Beginner" />
    <meta name="author" content="Krishna" />

    <!-- Social Preview Tags -->
    <meta property="og:title" content="Best Web Tutorial" />

üåê What are <meta> tags?

  • <meta> tags give extra info about the page.
  • They are useful for:
    • Search engines (SEO)
    • Social media previews (WhatsApp, Facebook)
    • Mobile responsiveness (viewport)

Example:

<meta name="description" content="Learn HTML the easy way!">
<meta name="keywords" content="HTML, Web development, Beginner">
